In this project, I developed an automated song segmentation system using the K-Means Clustering algorithm. After pre-processing the Spotify dataset and scaling the audio features, the model successfully grouped over 32,000 songs into 6 distinct clusters based on musical attributes like danceability, energy, and tempo. The correlation analysis revealed a strong link between loudness and energy, while the final cluster-genre visualization demonstrated how different musical styles share similar technical properties. This system provides the foundation for a song recommendation engine by suggesting tracks from the same cluster as the user's favorite songs.
